🚀 GUIDE DE LANCEMENT - APPLICATION ISESIR
========================================

MÉTHODE RECOMMANDÉE :
🎯 Double-cliquez sur "lancer_application.bat"

POURQUOI NE PAS double-cliquer sur app.py ?
❌ Double-cliquer sur app.py ouvre et ferme immédiatement la CMD
❌ Aucun message d'erreur visible
❌ Impossible de diagnostiquer les problèmes

AVANTAGES du fichier .bat :
✅ Vérification automatique de Python
✅ Installation automatique des dépendances
✅ Messages d'erreur détaillés et visibles
✅ Fenêtre qui reste ouverte pour diagnostics
✅ Instructions claires en cas de problème

ÉTAPES DE LANCEMENT :
1. 📁 Ouvrez le dossier de l'application
2. 🖱️ Double-cliquez sur "lancer_application.bat"
3. 🔄 Attendez le démarrage (installation automatique si nécessaire)
4. 🌐 Ouvrez votre navigateur sur http://localhost:5000

EN CAS DE PROBLÈME :
• La fenêtre CMD reste ouverte pour voir les erreurs
• Messages d'aide automatiques selon le problème
• Instructions de résolution incluses

ALTERNATIVE MANUELLE :
Si le fichier .bat ne fonctionne pas :
1. Ouvrez une invite de commande
2. Naviguez vers le dossier de l'application
3. Tapez : python app.py
4. Lisez les messages d'erreur s'il y en a

DÉPENDANCES REQUISES :
• Python 3.7+
• flask, pandas, xlrd, openpyxl
• matplotlib, seaborn, reportlab, pillow, werkzeug
