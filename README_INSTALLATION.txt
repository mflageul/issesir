INSTALLATION ET LANCEMENT :

1. PRÉREQUIS :
   - Python 3.7+ installé sur votre système
   - Connexion Internet pour les dépendances

2. INSTALLATION :
   - Extraire tous les fichiers dans un dossier
   - Ouvrir une console/terminal dans ce dossier
   - Exécuter : pip install flask pandas numpy matplotlib seaborn reportlab pillow openpyxl

3. LANCEMENT :
   - Windows : Double-clic sur LANCER_APPLICATION.bat
   - Autres : python app.py
   - Ouvrir http://localhost:5000 dans votre navigateur

FICHIERS INCLUS :
✓ app.py - Application Flask principale
✓ utils/ - Modules de traitement des données
✓ templates/ - Interface web HTML
✓ static/ - Ressources CSS/JS/Images
✓ LANCER_APPLICATION.bat - Script Windows
